LiteTorrent 
(EN)
A lightweight, modern Torrent Downloader built with Python, featuring a clean Dark Mode GUI.

LiteTorrent is a desktop application that allows you to download files via torrent files using the powerful libtorrent library and a user-friendly tkinter interface.
 Features

    Simple & Clean UI: Modern dark theme with real-time progress tracking.

    Detailed Torrent Info: View file name, total size, number of files, and trackers before starting.

    Live Statistics: Monitor download/upload speeds, peer count, and percentage progress.

    Download Controls: Start, Stop, Pause, and Resume functionality.

    Visual Feedback: Dynamic status bar and progress bar.
